## 🎯 시스템 개요

CAMS(Computerized Asset Management System)는 금형 개발부터 폐기까지 전체 Lifecycle을 QR 스캔과 GPS 위치 기반으로 관리하는 시스템입니다.

### 주요 특징
- **QR 코드 기반 빠른 접근**: 금형 QR 스캔으로 즉시 정보 조회 및 작업 수행
- **GPS 위치 추적**: 금형 위치 실시간 추적 (50m 오차 이내)
- **생산수량 기반 점검 관리**: 타수 자동 누적 및 점검 스케줄 자동 업데이트
- **실시간 알림**: 점검 예정, 수리 요청, 승인 요청 등 16종 알림

## 📱 주요 기능 사용법

### 1. 로그인
1. 시스템 접속 (PC 또는 모바일)
2. 사용자 ID/비밀번호 입력
3. 로그인 버튼 클릭
4. 역할에 따른 대시보드로 자동 이동

> **세션 유지**: 8시간 (자동 로그아웃)

### 2. QR 스캔 (모바일)
1. 모바일 홈에서 "QR 스캔" 버튼 클릭
2. 금형 QR 코드 스캔
3. 금형 상세 정보 확인
4. 원하는 작업 선택 (점검, 수리요청 등)

> **GPS 자동 기록**: QR 스캔 시 현재 위치 자동 저장

### 3. 일상점검 (생산처)
1. 금형 상세 → "일상점검" 클릭
2. 생산수량 입력 (필수)
3. 점검 항목 체크
4. 사진 첨부 (선택)
5. 제출

> **자동 연동**: 생산수량 → 타수 누적 → 점검 스케줄 업데이트 → 알람 생성

### 4. 정기점검 (생산처)
1. 금형 상세 → "정기점검" 클릭
2. 점검 차수 확인 (1차/2차/3차)
3. 체크리스트 항목별 PASS/FAIL 입력
4. NG 항목 발생 시 사유 입력
5. 제출 → 승인 대기

### 5. 수리 요청 (생산처)
1. 금형 상세 → "수리요청" 클릭
2. 수리 유형 선택 (긴급/일반)
3. 증상 설명 입력
4. 사진 첨부
5. 제출 → 제작처 접수

### 6. 금형 이관 (생산처 ↔ 제작처)
1. 금형 상세 → "이관관리" 클릭
2. 이관 유형 선택 (반출/입고)
3. 4M 체크리스트 작성
4. 반출/입고 체크리스트 작성
5. 승인 요청

## 🔔 알림 시스템

### 알림 유형
| 유형 | 설명 | 우선순위 |
|------|------|----------|
| 점검 예정 (타수) | 타수 90% 도달 시 | High |
| 점검 예정 (일자) | D-7, D-3, D-1 | Medium |
| 점검 지연 | 예정일 초과 | Urgent |
| 유지보전 예정 | 세척/습합 예정 | Medium |
| 수리 요청 | 신규 수리 요청 | High |
| 폐기 요청 | 폐기 승인 요청 | Medium |
| 이관 요청 | 이관 승인 요청 | Medium |
| 체크리스트 알림 | 제출/승인/반려 | Low |

### 알림 확인
- **PC**: 상단 헤더 알림 아이콘 클릭
- **모바일**: 하단 "알림" 메뉴 클릭
- **읽음 처리**: 알림 클릭 시 자동 읽음 처리

## ⚠️ 주의사항

### 보안
- 비밀번호는 8자 이상, 영문+숫자+특수문자 조합
- 로그인 5회 실패 시 계정 잠금 (관리자 해제 필요)
- 공용 PC 사용 후 반드시 로그아웃

### 데이터 입력
- 생산수량은 정확히 입력 (타수 자동 계산에 영향)
- 점검 항목 누락 시 제출 불가
- 사진 첨부 시 파일 크기 10MB 이하

### 모바일 사용
- QR 스캔 시 카메라 권한 허용 필요
- GPS 위치 서비스 활성화 필요
- 안정적인 네트워크 연결 권장
