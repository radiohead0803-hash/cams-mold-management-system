# 정기점검 체크리스트 UI 설계

## 🔄 일상점검과 동일한 기본 금형 정보

정기점검도 일상점검과 동일하게 QR/마스터 정보를 기반으로 자동 입력합니다.
- **품번** (QR 스캔 또는 금형 마스터) → 예: `M-2024-001`
- **품명** (마스터) → 예: `스마트폰 케이스 금형`
- **현재 위치** (저장된 창고 또는 작업장) → 예: `A구역-01`
- **누적 쇼트수** (마지막 일상/정기점검 데이터) → 예: `9,500 회`
- **점검 시작 시간** (자동 시스템 시간)
- **GPS 위치** (정기점검 대상 장소에서 재차 캡처)

이 정보는 점검 시작 화면 상단에 카드 형태로 노출되며, 작업자가 변경하지 않아도 항상 최신 상태로 유지됩니다.

```
┌────────────────────────────────────────┐
│  정기점검                               │
│  품번: M-2024-001                       │
│  품명: 스마트폰 케이스 금형             │
│  위치: A구역-01                         │
│  누적 쇼트수: 9,500 회                  │
│  GPS: 37.5665° N / 126.9780° E          │
│  시작 시간: 2025-11-20 09:30             │
└────────────────────────────────────────┘
```

자동 입력 정보를 기준으로 정기점검은 1차(100K), 2차(500K), 3차(1M) 등의 주기에 따라 세분화된 카테고리를 확인합니다.

## 🧰 정기점검 항목 구성

### 1. 정기점검 공통 (모든 주기)
- 장비 작동 확인 (슬라이드, 가이드, 밀핀 등)
- 냉각라인 및 수리 상태
- 외관/도면 표시 부위의 이상 여부
- 예방차원의 윤활 상태
- 주요 치수 및 변형 감지

### 2. 1차 정기점검 (100K)
- 외관 분해 없이 상태 확인 (외관/필러/가이드)
- 금형 체결 상태, 로케이션 시스템 확인
- 온도·압력 센서 모니터링
- 냉각수 흐름 ○/×, 유량/압력

- **청소/습합 보강**: 벤트/게이트부, 습합 접합면 정리, 금형 외곽 세척 포함
- **알림**: 80,000 SHOT 진입 예정 시 세척/습합 미리 예약
### 3. 2차 정기점검 (500K)
- 금형 일부 분해 포함(슬라이드, 게이트)
- 표면 상태, 심부러짐/마모 여부
- 취출 시스템(핀/스크루) 성능 테스트
- 정밀 윤활(특정 그리스/오일) 재적용
- 치수측정(게이트 대칭, 핵심 치수)

- **전면 세척/습합 점검**: 분해 후 세척, 습합 간극 측정 포함
- **알림**: 200K까지 세척/습합 스케줄 기록
### 4. 3차 정기점검 (1M)
- 전체 분해 및 도면 검토
- 외관/명판/로고 훼손 여부
- 전기/배선/히터/센서 전수 점검
- 냉각계통 라인 전체 교체/세척 여부
- 방청·보관 처리 준비

### 5. 비가동시 습합/세척 연계
- 습합점검: 슬라이드/리프터/릴리즈 림 간극, 접촉상태
- 세척점검: 금형 전체 세척, 방청유 도포, 마모 부위 집중 점검

### 6. 습합점검 체크리스트 (이상점검과 동일한 기본 정보)
- **기본 금형 정보**: QR/마스터 기반 자동 입력 (품번, 품명, 위치, 누적 타수, GPS, 시작시간)
- **점검 대상**: 습합(접합) 부위, 가이드핀/리프트핀, 슬라이드, 금형 외곽
- **입력 형식**: 양호/주의/불량 버튼 + 특이사항 텍스트 + 사진 업로드
- **이상 조건**: 틈새, 쇼트 및 충격, 누수, 윤활 부족 등
- **가이드 링크**: 이상점검 도움말 팝업과 동일한 요약 가이드 제공
- **결과 요약**: 양호/정비 필요/수리 필요 버튼과 경고 메시지(필수 항목 누락/반복 이상)

💡 동일한 방식으로 진행되므로 습합점검에서는 이전 일상점검·정기점검 히스토리를 참조해 반복 이상을 체크하고, 문제가 있으면 `issue` 기록과 `maintenance_requests` 등록을 자동화합니다.

💡 **자동 알림**: 90% 도달 시 모바일/PC 알림과 이메일/LINE 등 채널로 발송하여 놓치는 일이 없도록 합니다.

## 📌 20,000 SHOT(3개월) 주요 점검 항목

```
┌────────────────────────────────────────────────────┐
│  선택된 점검: 20,000 SHOT 점검                       │
│  필수 항목: 1. 파팅면/성형면 | 2. 벤트/게이트 | 3. 작동부 | │
│             4. 습합(접합) | 5. 취출계통 | 6. 냉각/유압 | │
├────────────────────────────────────────────────────┤
│  점검 결과 종합                                        │
│  [양호] 정상 사용                        [정비 필요] 예방 정비│
│  [수리 필요] 즉시 수리 요청                                 │
└────────────────────────────────────────────────────┘
```

### 20,000 SHOT 점검 항목 요약
- **파팅면/성형면**: 파팅면 단차 ±0.02mm, 성형면 손상/표면 이상 여부
- **벤트/게이트**: 벤트홀 막힘, 게이트 청결, 0.03mm 이상 마모시 재가공
- **작동부**: 슬라이드 이상음/걸림/노유 여부, 원활한 작동 확인
- **습합(접합)**: 금형 간극, 단차, 접합 정렬 상태 ±0.02mm 이내 유지
- **취출계통**: 밀핀/스프링 작동, 박힘/변형/마모 없음
- **냉각/유압 연결부**: 누유/누수 여부, 조인트·커넥터·호스 상태

💡 **결과 버튼 구성**: 작업자는 점검 후 ‘양호’, ‘정비 필요’, ‘수리 필요’ 버튼을 눌러 상태를 기록하고, 필요시 정비 요청을 즉시 생성합니다.

🧼 **세척제 선택 (모든 정기점검 단계)**: 작업 조건에 따라 드라이아이스, 초음파 세척, 특수 케미컬 등을 선택하고 사용한 세척제 + 희석 비율을 점검표에 기록합니다. 50K/80K에서는 최신 세척제를 재평가하고, 100K 이상에서는 기본 선택을 검증한 다음 재사용합니다.

## 📌 50,000 SHOT 점검 항목 (20,000 SHOT + 추가 항목)

```
┌────────────────────────────────────────────────────┐
│  선택된 점검: 50,000 SHOT 점검                       │
│  구성: 20,000 SHOT 필수 항목 전부 + 아래 추가 항목      │
├────────────────────────────────────────────────────┤
│  • 벤트/게이트부 마모, 청결                      │
│  • 작동부(가이드핀/리테이너, 리프트핀·엘글라)     │
│  • 냉각라인 유량 및 온도 편차                      │
│  • 히터·센서·배선 접촉 상태                         │
│  • 표면처리 상태(크롬층/코팅)                       │
│  • 습합 정확도 및 틈새 유지                        │
│  • 취출핀 마모, 볼트너트/볼트게이트 접속상태        │
└────────────────────────────────────────────────────┘
```

- **벤트·게이트부 마모**: 핀/인서트 마모상태 관찰, 마모 0.05mm 이상이면 교체 또는 조정
- **가이드핀/리테이너, 리프트핀/엘글라**: 마모/유격 ±0.02mm, 변형·이상음 없음
- **냉각라인 유량/온도**: 유량 저하·막힘 확인, 온도 편차 ±10% 이내, 누수 없음
- **히터/센서/배선**: 히터 저항 ±10% 이내, 온도센서·배선 손상·접촉불량·이상 징후 확인
- **표면처리(코팅/크롬층)**: 박리, 변색, 두께 이상 여부 및 도포 균일성 확인
- **습합 정확도**: 가이드 핀 기준 수평각 ±0.02mm, 간극/접합력 확인
- **취출핀/볼트너트 점검**: 핀 마모·손상 여부, 볼트너트ㆍ게이트 작동 상태와 배선 접점 상태

**추가 팁**: 50,000 SHOT 점검에서는 20,000 SHOT 점검 결과를 통한 히스토리와 비교하여 마모 추이를 기록하고, 필요시 100,000 SHOT 점검에서의 전체 분해 계획에 반영합니다.

### 80,000 SHOT 추가 항목 요약
- **외곽/내부 세척**: 캐비티·코아·게이트 주변 분진, 코팅 잔존물 제거
- **세척제 선택**: 드라이아이스, 초음파, 특수 케미컬 등 작업 조건에 따라 선택하며, 해당 세척제 및 희석 비율을 기록
- **습합 정렬**: 서로 접합되는 면의 틈새, 가이드핀 각도, 습합 압력 균일성 확인
- **냉각·유압 클리닝**: 필터 교체 권고, 누수 체크, 호스/커넥터 세척
- **염화/세척 기록**: 세척 완료 시간/담당/소요 타수 기록 유지
- **이상감지 알림**: 80K 달성 시 관리자에게 자동 알림+정비권장(수리 시그널)

🔁 **중복 관리**: 80K는 50K에 비해 청소·습합 이중 확인이 강조되므로 50K 체크항목과 겹치는 내용은 이전 결과 참조로 표기하고, 전체 세척/습합 확인은 새롭게 기록합니다.

- **냉각라인 스케일**: 스케일 제거 및 이물 세척, 냉각수흐름 확보, 틈새 청소

- **표면/코팅**: 코팅 박리, 크롬층 불균일, 변색 확인 및 재코팅 여부 판단
- **볼트너트/게이트 연계**: 볼트너트/게이트 작동 후 판탈림 여부, 배선 절연 손상 체크

- **정밀 윤활 및 마모 분석**: 마모 예측치, 립/핀 마모 기록, 예상 교체 시점 산정

중복 정리 팁: 각 단계에서는 이전 SHOT 결과를 기준으로 중복된 항목을 패스하되, 마모 또는 이상이 반복되는 경우 100,000 SHOT에서 충분히 기록하고 보완 조치를 명기하여 후속 수리로 넘어가도록 합니다.

## ## 800,000 SHOT 점검 요약 (기존 항목 + 통합 청소/습합)
- 기준 폭탄: 800K 이상 진입 시 기존 20K/50K/100K 항목 모두 반영하면서 다음 내용을 추가합니다.
  - 파팅면 오염 여부 (Gas·오일·수지 찌꺼기) 점검 및 Flash 방지 작업
  - 성형면 오염/스크래치 제거 및 외관 마모 확인
  - 슬라이드/핀 주변 이물 제거 (가동 전 청소 포함)
  - 가이드핀·앵글리핀 윤활 상태 확인 및 필요시 리필
  - 냉각라인 전체 세척 및 냉각효율 확인
  - 세척제/세척방법 기록 (드라이아이스, 초음파, 특수 케미컬 등 포함)
  - 윤활 상태 저장 및 이력화하여 이상 반복 시 alert
  - 체크리스트 결과(양호/정비 필요/수리 필요) 및 issue 생성으로 추적

비고: 800K에서는 '정기점검 공통' 항목과 전체 세척/습합을 통합하며, 이상 부위는 issue/maintenance_requests에 자동으로 연결되도록 합니다.

## 120K 이상(120K / 150K / 200K) 점검 구조

120K 이상 구간에서는 기존 100K 점검 항목을 그대로 유지하면서 주기별 증분을 별도로 구성하지 않고 동일한 체크리스트를 반복 사용합니다. 단, 누적 마모/비정상 징후는 매 회차마다 비교하여 증가 추세를 기록하고, 실시간 알림에 다음 주기(예: 150K, 200K) 진입 시점을 업데이트합니다.

### 운영 팁
- 120K/150K/200K 스케줄: 대시보드에 각 목표 SHOT별 마일스톤(예: 120K: 2개월, 150K: 3개월, 200K: 6개월) 표시 후 100K 점검 완료 이후 자동으로 다음 단계로 전환
- 중복 조회 정책: 과거 100K 점검일과 비교하여 워크북에서 동일한 항목을 재기록할 필요는 없지만, 이상 징후(냉각 누수, 핀 마모, 코팅 박리 등)는 issue 테이블에 기록 후 maintenance_requests 로 즉시 연결
- 추적 알림: 120K/150K/200K 경고는 현재 SHOT 대비 90% 도달 시 자동 발생, 200K 이후에는 반드시 분해/리빌트 계획 포함
- **중복 조회 정책**: 과거 100K 점검일과 비교하여 워크북에서 동일한 항목을 재기록할 필요는 없지만, 이상 징후(냉각 누수, 핀 마모, 코팅 박리 등)는 `issue` 테이블에 기록 후 `maintenance_requests` 로 즉시 연결
- **추적 알림**: 120K/150K/200K 경고는 현재 SHOT 대비 90% 도달 시 자동 발생, 200K 이후에는 반드시 분해/리빌트 계획 포함

## ��📱 모바일 UI 흐름
1. **기본 정보 자동 입력** → QR/마스터 데이터
2. **GPS 위치 캡처** (필수) → 정기점검 시작 버튼 활성화
3. **주기 선택** (1차, 2차, 3차) → 해당 항목 그룹 표시
4. **항목별 상태 입력** (양호/주의/불량) + 특이사항 + 사진
5. **특별 조치 로그** (작업 지시/수리 요청)
6. **완료 보고서** → 작업자 서명 + 관리자 승인
