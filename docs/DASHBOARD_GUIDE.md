## 👥 사용자 유형별 대시보드 개요

### 4가지 사용자 유형과 대시보드 구성

```
┌─────────────────────────────────────────────────────────────┐
│              사용자 유형별 대시보드 매핑                       │
│                                                               │
│  1️⃣ CAMS 시스템 관리 담당 (본사)                              │
│     → 시스템 관리 대시보드                                     │
│     • 전사 통합 현황 모니터링                                  │
│     • 사용자/권한/시스템 설정 관리                             │
│     • 실시간 알람 및 GPS 추적                                  │
│                                                               │
│  2️⃣ 금형개발 담당 (본사)                                      │
│     → 금형 관리 대시보드                                       │
│     • 금형 생명주기 관리                                       │
│     • 승인/검토 워크플로우                                     │
│     • 제작처 성과 모니터링                                     │
│                                                               │
│  3️⃣ 금형제작처 (Maker)                                        │
│     → 제작처 작업 대시보드                                     │
│     • 제작/수리 작업 현황                                      │
│     • QR 코드 관리                                             │
│     • 시운전 및 귀책 협의                                      │
│                                                               │
│  4️⃣ 생산처 (Plant)                                           │
│     → 생산처 작업 대시보드                                     │
│     • 점검 일정 및 현황                                        │
│     • QR 스캔 기반 작업                                        │
│     • 수리 요청 및 이관 관리                                   │
└─────────────────────────────────────────────────────────────┘
```

### 로그인 후 자동 라우팅

```javascript
// 사용자 유형에 따른 대시보드 자동 이동
const dashboardRoutes = {
  'system_admin': '/dashboard/system-admin',
  'mold_developer': '/dashboard/mold-developer',
  'maker': '/dashboard/maker',
  'plant': '/dashboard/plant'
};

// JWT 토큰에서 user_type 추출 후 자동 리다이렉트
const userType = jwt.decode(token).user_type;
router.push(dashboardRoutes[userType]);
```

## 🖥️ CAMS 시스템 관리 담당 대시보드

### 개요
CAMS 시스템 관리 담당은 전사 시스템을 총괄 관리하며, 모든 금형과 협력사의 활동을 실시간으로 모니터링합니다.

### 주요 기능

#### 1. 전사 통합 현황
```
┌─────────────────────────────────────────────────────────────┐
│                    전사 통합 대시보드                          │
│                                                               │
│  📊 금형 현황 요약                                             │
│  ┌──────────────┬──────────────┬──────────────┐             │
│  │ 전체 금형    │ 양산 중      │ 수리 중      │             │
│  │    245개     │    198개     │     12개     │             │
│  └──────────────┴──────────────┴──────────────┘             │
│                                                               │
│  🔔 실시간 알람 (최근 24시간)                                  │
│  • Critical: 3건                                              │
│  • Major: 12건                                                │
│  • Minor: 45건                                                │
│                                                               │
│  📍 GPS 위치 추적                                              │
│  • 등록된 위치: 198개                                          │
│  • 위치 이탈: 2개                                              │
│                                                               │
│  ⚙️ 시스템 상태                                                │
│  • 활성 사용자: 156명                                          │
│  • 금일 QR 스캔: 1,234회                                       │
└─────────────────────────────────────────────────────────────┘
```

#### 2. 시스템 설정 및 관리
- **사용자 관리**
  - 사용자 등록/수정/삭제
  - 비밀번호 초기화
  - 계정 잠금/해제
  
- **권한 관리**
  - 권한 그룹 설정
  - 메뉴별 권한 설정
  
- **점검표 관리**
  - 일상점검 체크리스트
  - 정기점검 체크리스트
  - 버전 관리
  
- **알람 기준 설정**
  - 타수 기준 설정
  - NG 기준 설정
  - 위치 이탈 기준

#### 3. 실시간 모니터링
```
┌─────────────────────────────────────────────────────────────┐
│                  실시간 활동 피드                              │
│                                                               │
│  🔴 [09:30] Critical NG 발생                                  │
│     금형: M2024-001 | 생산처: A공장                           │
│     → 즉시 생산중단 조치 필요                                  │
│                                                               │
│  🟡 [09:25] 정기점검 지연                                      │
│     금형: M2024-045 | 생산처: B공장                           │
│     → 예정일 초과 3일                                          │
│                                                               │
│  🟢 [09:20] 수리 완료                                          │
│     금형: M2024-023 | 제작처: C제작소                         │
│     → 품질 확인 후 정상화                                      │
└─────────────────────────────────────────────────────────────┘
```

## � QR 스캔 후 모바일 대시보드

### 개요
QR 코드 스캔 후 사용자 유형에 따라 최적화된 모바일 대시보드가 표시됩니다. 모바일 환경에 맞춰 터치 친화적이고 직관적인 UI를 제공합니다.

### QR 스캔 프로세스

```
┌─────────────────────────────────────────────────────────────┐
│                  QR 스캔 프로세스                              │
│                                                               │
│  [1단계] QR 코드 스캔                                          │
│         ↓                                                     │
│  [2단계] 금형 정보 로드 + GPS 위치 확인                        │
│         ↓                                                     │
│  [3단계] 사용자 유형 확인 (JWT 토큰)                           │
│         ↓                                                     │
│  [4단계] 유형별 모바일 대시보드 표시                           │
│         ↓                                                     │
│         ├─ 제작처 → 제작/수리 작업 메뉴                        │
│         └─ 생산처 → 점검/생산수량 입력 메뉴                    │
└─────────────────────────────────────────────────────────────┘
```
