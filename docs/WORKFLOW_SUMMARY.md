# 금형 관리 시스템 업무 플로우 종합 요약

## 📋 문서 개요

이 문서는 금형 관리 전산시스템의 전체 업무 플로우를 종합적으로 요약한 문서입니다.

---

## 👥 사용자 유형 4가지 (공통 정의)
- **CAMS 시스템 관리 담당 (본사)** – 시스템 규칙 관리, 전사 모니터링, 권한 설정/감사
- **금형개발 담당 (본사)** – 금형 사양 정의·승인, 제작처 관리, 전사 통계/사용 기준
- **금형제작처 (Maker)** – 설계/제작/시운전/수리, QR 코드 출력/부착, 현장 작업 실행
- **생산처 (Plant)** – 양산/점검/수리요청/이관, QR 스캔 기반 작업 및 생산수량 입력

---

## 🎯 시스템 핵심 개념

### 4단 역할 구조

```
┌─────────────────────────────────────────────────────────────┐
│              CAMS 시스템 관리 담당 (본사)                     │
│  - 시스템 규칙 관리 / 전사 모니터링 / 권한관리              │
└─────────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────────┐
│                금형개발 담당 (본사)                           │
│  - 금형사양 정의/승인 / 제작처 관리                         │
└─────────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────────┐
│                  금형제작처 (Maker)                           │
│              담당자 / 작업자 (동일 ID 사용)                    │
│  - 설계 / 제작 / 시운전 / 수리                                │
└─────────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────────┐
│                   생산처 (Plant)                             │
│              담당자 / 작업자 (동일 ID 사용)                    │
│  - 양산 / 점검 / 수리요청 / 이관                              │
└─────────────────────────────────────────────────────────────┘
```

### QR + GPS 기반 운영 원칙

- **모든 작업은 QR 스캔으로 시작**
- **GPS 자동 기록** (작업자 QR 스캔 단말 기준: 헤드폰 스캐너/모바일 기기)
- **권한별 메뉴 자동 구분**
- **실시간 데이터 동기화**

## 🔄 금형 Lifecycle 10단계

### 1. 금형 개발 (금형개발 담당 + CAMS 시스템 관리 담당)
- 부품 기획 / 금형 사양 정의
- 금형 제작처 선정
- 금형코드 생성
- **금형정보 최초 등록 시 QR 코드 자동 생성**

### 2. 금형 설계 (제작처 + 본사)
- 2D/3D 설계, 구조 검토
- 설계 변경 / 승인

### 3. 금형 제작 (제작처)
- **자동 생성된 QR 코드 확인 및 출력**
- 가공 / 조립 / 핫러너 / 기능부 제작
- 제작 진행률 관리
- **제작 완료 후 QR 코드 금형 명판에 부착**
  - 주 부착 위치: 금형 상판
  - 보조 부착 위치: 금형 측면
  - 방수/방유/내열 라벨 사용

### 4. 시운전 (Try-out)
- 초도품 성형 테스트
- 품질 / 치수 / 사이클 검증
- PASS/FAIL 판정

### 5. 양산 이관 (제작처 → 생산처)
- 출고 / 운송 / 입고
- 금형 상태 확인 / 설치
- 금형 마스터 자동 생성

### 6. 양산 생산 (생산처)
- 일상점검 (QR + GPS + 생산수량 입력)
- 정기점검 (1차: 100K, 2차: 500K, 3차: 1M)
- 타수 자동 업데이트
- 점검 스케줄 자동 계산

### 7. 금형 수리 (생산처/제작처)
- 수리 요청 / 수리 진행
- 단계별 사진 (수리 전/중/후)
- 수리 완료 후 보고

### 8. 금형 유지보전
- 품질 저하 예방
- 금형 성능 유지
- 정기 세척 / 습합

### 9. 금형 이관 (공장 간 / 외주)
- 반출 / 입고 체크리스트
- GPS 기반 위치 추적
- 이관 사유 및 목적지 기록

### 10. 금형 폐기 (금형개발 담당 승인)
- 타수 초과 / 기능 상실
- 고객 승인 필요
- 폐기 사진 및 기록 저장

## 🚨 NG(불합격) 자동 연계 프로세스

### NG 유형 분류

```
┌─────────────────────────────────────────────────────────────┐
│  Minor (경미)    → 협력사 자체 조치                          │
│  Major (중요)    → 수리 요청 필요                            │
│  Critical (치명) → 생산중단 + 본사 즉시 알림                 │
└─────────────────────────────────────────────────────────────┘
```

### 자동 연계 5단계

1. **NG 유형 자동 분류**
2. **자동 생성 데이터** (알람, 점검 결과, 수리 요청, 금형상태 변경)
3. **자동 알림 발송** (생산처, 제작처, 본사)
4. **조치 프로세스 자동 전개** (Minor/Major/Critical별 대응)
5. **조치 완료 후 자동 반영** (상태 정상화, 스케줄 재계산)

## 📅 정기점검 시스템

### 정기점검 기준

- **1차 점검**: 100,000 타 (100K)
- **2차 점검**: 500,000 타 (500K)
- **3차 점검**: 1,000,000 타 (1M)

### 체크리스트 5개 카테고리

1. **외관 점검** - 균열 / 마모 / 부식 / 오염
2. **기능 점검** - 슬라이드 / 리프터 / 이젝터 / 냉각
3. **치수 점검** - 제품 치수 측정
4. **성능 점검** - 사이클 타임 / 불량률 / 온도
5. **문서 점검** - 이력 확인 / 수리 기록 / 변경 사항

### 자동 연계

- **PASS** → 다음 점검 일정 자동 생성
- **FAIL** → NG 프로세스 자동 연동
- **Critical FAIL** → 생산중단 + 본사 긴급 알림

## 🛰️ QR 시스템 운영

### QR 코드 생성 및 부착

```
[1단계] 금형개발 담당이 금형정보 최초 등록
         ↓
[2단계] 시스템이 자동으로 고유 QR 코드 생성
         ↓
[3단계] 제작처로 QR 코드 자동 배포
         ↓
[4단계] 제작처에서 QR 코드 출력 (내구성 라벨)
         ↓
[5단계] 금형 제작 완료 후 명판에 부착
```

### QR 코드 구조

```
CAMS-MOLD-[금형코드]-[체크섬]
예: CAMS-MOLD-M2024001-A3F7

※ 금형개발 담당이 금형정보 등록 시 시스템이 자동 생성
※ 제작처로 자동 배포되며, 제작처에서 명판에 부착
```

### QR 스캔 프로세스

1. **QR 스캔** → 금형정보 자동 로드
2. **권한 확인** → 메뉴 자동 구분
3. **작업 선택** → 일상/정기점검, 수리, 이관 등
4. **GPS 기록** → 위치 자동 저장
5. **데이터 동기화** → 실시간 연동

### QR 세션 관리

- **유효 시간**: 8시간
- **자동 기록**: GPS 위치, 사용자 ID, 스캔 시간
- **활용**: 작업 이력 추적, 위치 이동 감지

## 📊 데이터 흐름

### 본사 → 제작처 → 생산처

```
본사 (HQ)
  ↓ 금형 사양 전달
제작처 (Maker)
  ↓ 제작완료 → 금형마스터 자동 생성
생산처 (Plant)
  ↓ 점검/수리/타수 정보
본사 (HQ) 대시보드 통합 모니터링
```

### 실시간 동기화 항목

- 금형 상태 변경
- 점검 결과 (PASS/FAIL)
- 생산수량 및 타수
- 알람 발생
- 수리 요청 및 진행 상태
- GPS 위치 정보

## 🎯 핵심 기능 요약

### 1. QR + GPS 기반 현장 작업
- QR 코드 스캔으로 빠른 접근 (1초)
- GPS 위치 자동 기록 (50m 오차 이내)
- 8시간 세션 유지

### 2. 생산수량 기반 점검 관리
- 일상점검 시 생산수량 필수 입력
- 자동 누적 계산
- 점검 스케줄 자동 업데이트
- 타수 기록 자동 업데이트
- 알람 자동 생성

### 3. NG 자동 연계 시스템
- NG 유형 자동 분류
- 조치 프로세스 자동 전개
- 실시간 알림 발송
- 상태 자동 업데이트

### 4. 정기점검 자동 관리
- 타수 기반 점검 일정 자동 계산
- 표준 체크리스트 자동 로드
- PASS/FAIL 자동 연계
- 다음 점검 일정 자동 생성

### 5. 전사 실시간 연동
- 본사-제작처-생산처 실시간 데이터 동기화
- 통합 대시보드 모니터링
- GPS 기반 금형 위치 추적
- 협력사별 성과 분석
