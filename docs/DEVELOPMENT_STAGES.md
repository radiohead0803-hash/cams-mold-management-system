# 개발 진행 단계별 가이드

## 📋 전체 개발 단계

### Phase 1: 기반 구축 (Week 1)
- 데이터베이스 설계 및 구현 (50개 테이블)
- JWT 인증 시스템
- Express API 기본 구조

### Phase 2: 핵심 기능 (Week 2)
- QR 코드 스캔 시스템
- GPS 위치 추적
- 금형 관리 CRUD
- 점검 시스템 기본

### Phase 3: 협력사 기능 (Week 3)
- 일상점검 + 생산수량 입력
- 정기점검 (1차/2차/3차)
- 수리 요청 및 관리
- 이관 관리

### Phase 4: 관리자 기능 (Week 4)
- 관리자 대시보드
- 본사 담당자 대시보드
- 승인 시스템
- 통계 및 리포트

### Phase 5: 최적화 및 배포 (Week 5)
- 성능 최적화
- 모바일 최적화
- 테스트 및 버그 수정
- 배포 및 운영 준비
