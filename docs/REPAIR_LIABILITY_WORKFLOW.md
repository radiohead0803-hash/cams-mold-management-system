## 🤝 귀책 협의 개요

### 귀책 협의 필요성

금형 수리 발생 시 수리 비용 부담 주체를 명확히 하기 위해 생산처, 금형제작처, 본사 간 귀책 협의 프로세스가 필요합니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    귀책 협의 목적                             │
│                                                               │
│  • 수리 비용 부담 주체 명확화                                  │
│  • 공정한 책임 분담                                            │
│  • 분쟁 최소화                                                 │
│  • 재발 방지 대책 수립                                         │
└─────────────────────────────────────────────────────────────┘
```

## 🔄 귀책 협의 프로세스

### 전체 흐름도

```
[수리 요청 발생]
         ↓
[1차 협의] 생산처 ↔ 금형제작처
         ↓
    합의 성공?
    ↙        ↘
  YES         NO
   ↓           ↓
[수리 진행]  [2차 협의]
             본사 개입
                ↓
           [최종 판정]
                ↓
           [수리 진행]
```

### 1차 협의: 생산처 ↔ 금형제작처

```
┌─────────────────────────────────────────────────────────────┐
│          1차 협의: 생산처 ↔ 금형제작처 직접 협의              │
│                                                               │
│  [Step 1] 수리 원인 분석                                      │
│  • 생산처: NG 발생 상황 및 원인 설명                           │
│    - 발생 시점, 작업 조건                                      │
│    - 사진 및 증빙 자료                                         │
│                                                               │
│  • 제작처: 금형 상태 및 이력 검토                              │
│    - 제작 사양 확인                                            │
│    - 이전 수리 이력 확인                                       │
│    - 사용 기간 및 타수 확인                                    │
│                                                               │
│  [Step 2] 귀책 당사자 1차 판단                                 │
│  • 양측 의견 교환                                              │
│  • 증빙 자료 공유                                              │
│  • 귀책 비율 논의                                              │
│                                                               │
│  [Step 3] 협의 결과 시스템 기록                                │
│  • repair_liability 테이블에 기록                             │
│    - liability_party: 귀책 당사자                             │
│    - liability_ratio: 귀책 비율 (%)                           │
│    - agreement_status: 'agreed' / 'disagreed'                │
│    - notes: 협의 내용                                          │
│                                                               │
│  [Step 4] 합의 여부 판단                                       │
│  • 합의 성공 → 수리 진행                                       │
│  • 합의 실패 → 2차 협의 (본사 개입)                            │
└─────────────────────────────────────────────────────────────┘
```

### 2차 협의: 본사 개입

```
┌─────────────────────────────────────────────────────────────┐
│            2차 협의: 본사 개입 (합의 불가 시)                  │
│                                                               │
│  [Step 1] 본사 검토 요청                                       │
│  • 금형개발 담당에게 검토 요청                                  │
│  • 1차 협의 내용 전달                                          │
│  • 관련 자료 제출                                              │
│                                                               │
│  [Step 2] 금형개발 담당 분석                                   │
│  • 금형 사양 검토                                              │
│    - 설계 도면 확인                                            │
│    - 제작 사양서 확인                                          │
│    - 재질 및 강도 확인                                         │
│                                                               │
│  • 사용 이력 분석                                              │
│    - 누적 타수 확인                                            │
│    - 생산 조건 확인                                            │
│    - 일상/정기점검 이력 확인                                   │
│                                                               │
│  • 수리 이력 분석                                              │
│    - 과거 수리 내역 확인                                       │
│    - 동일 부위 반복 수리 여부                                  │
│    - 수리 주기 분석                                            │
│                                                               │
│  [Step 3] 최종 귀책 판정                                       │
│  • 객관적 증거 기반 판단                                       │
│  • 귀책 당사자 및 비율 결정                                    │
│  • 판정 사유 명시                                              │
│                                                               │
│  [Step 4] 비용 처리 방안 결정                                  │
│  • 귀책 비율에 따른 비용 분담                                  │
│  • 청구 방법 및 일정 결정                                      │
│  • 재발 방지 대책 수립                                         │
│                                                               │
│  [Step 5] 결과 통보                                            │
│  • 생산처 및 제작처에 결과 통보                                │
│  • 시스템에 최종 판정 기록                                     │
│  • 수리 진행 승인                                              │
└─────────────────────────────────────────────────────────────┘
```

## 📊 귀책 유형 분류

### 귀책 유형별 기준

```
┌─────────────────────────────────────────────────────────────┐
│  1. 생산처 귀책 (Plant Liability)                             │
│  ───────────────────────────────────────────────────────────│
│  원인:                                                         │
│  • 부적절한 사용                                               │
│    - 권장 조건 초과 사용                                       │
│    - 과도한 압력/온도 사용                                     │
│    - 부적합한 원료 사용                                        │
│                                                               │
│  • 관리 소홀                                                   │
│    - 일상점검 미실시                                           │
│    - 정기점검 지연                                             │
│    - 세척/습합 미실시                                          │
│                                                               │
│  • 작업자 과실                                                 │
│    - 부주의한 취급                                             │
│    - 충격/낙하                                                 │
│                                                               │
│  비용 부담: 생산처 100%                                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  2. 제작처 귀책 (Maker Liability)                             │
│  ───────────────────────────────────────────────────────────│
│  원인:                                                         │
│  • 설계 결함                                                   │
│    - 구조적 문제                                               │
│    - 강도 계산 오류                                            │
│    - 냉각 설계 불량                                            │
│                                                               │
│  • 제작 결함                                                   │
│    - 가공 불량                                                 │
│    - 조립 불량                                                 │
│    - 용접 불량                                                 │
│                                                               │
│  • 재질 불량                                                   │
│    - 부적합한 재질 사용                                        │
│    - 재질 강도 미달                                            │
│    - 열처리 불량                                               │
│                                                               │
│  비용 부담: 제작처 100%                                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  3. 자연 마모 (Natural Wear)                                  │
│  ───────────────────────────────────────────────────────────│
│  원인:                                                         │
│  • 정상 사용 범위 내 마모                                      │
│    - 목표 타수 달성 후 마모                                    │
│    - 예상 수명 범위 내 마모                                    │
│    - 정상적인 부품 노화                                        │
│                                                               │
│  • 소모성 부품 교체                                            │
│    - 이젝터 핀                                                 │
│    - 스프링                                                    │
│    - 씰 부품                                                   │
│                                                               │
│  비용 부담: 본사 또는 협의                                     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  4. 기타 (Others)                                             │
│  ───────────────────────────────────────────────────────────│
│  원인:                                                         │
│  • 불가항력                                                    │
│    - 자연재해                                                  │
│    - 화재/침수                                                 │
│                                                               │
│  • 외부 요인                                                   │
│    - 원료 불량 (공급사 책임)                                   │
│    - 설비 문제                                                 │
│                                                               │
│  비용 부담: 개별 협의                                          │
└─────────────────────────────────────────────────────────────┘
```

### 비용 청구 프로세스

```
[수리 완료]
     ↓
[비용 산정]
     ↓
[귀책 비율 적용]
     ↓

     ↓

```

### 데이터베이스 테이블

```sql
-- 귀책 협의 테이블
CREATE TABLE repair_liability (
    id SERIAL PRIMARY KEY,
    repair_id INTEGER REFERENCES repairs(id),
    
    -- 1차 협의
    first_negotiation_date TIMESTAMP,
    plant_opinion TEXT,
    maker_opinion TEXT,
    first_agreement_status VARCHAR(20), -- 'agreed', 'disagreed'
    
    -- 2차 협의 (본사 개입)
    second_negotiation_date TIMESTAMP,
    hq_reviewer_id INTEGER,
    hq_analysis TEXT,
    
    -- 최종 판정
    liability_party VARCHAR(50), -- 'plant', 'maker', 'natural_wear', 'shared', 'other'
    liability_ratio_plant INTEGER, -- 생산처 귀책 비율 (%)
    liability_ratio_maker INTEGER, -- 제작처 귀책 비율 (%)
    liability_ratio_hq INTEGER,    -- 본사 귀책 비율 (%)
    
    final_decision TEXT,
    decision_reason TEXT,
    
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 수리 비용 테이블
CREATE TABLE repair_costs (
    id SERIAL PRIMARY KEY,
    repair_id INTEGER REFERENCES repairs(id),
    liability_id INTEGER REFERENCES repair_liability(id),
    
    -- 비용 항목
    parts_cost DECIMAL(10,2),      -- 부품 비용
    labor_cost DECIMAL(10,2),      -- 인건비
    transport_cost DECIMAL(10,2),  -- 운송비
    downtime_cost DECIMAL(10,2),   -- 생산중단 손실
    other_cost DECIMAL(10,2),      -- 기타 비용
    total_cost DECIMAL(10,2),      -- 총 비용
    
    -- 비용 분담
    plant_cost DECIMAL(10,2),      -- 생산처 부담
    maker_cost DECIMAL(10,2),      -- 제작처 부담
    hq_cost DECIMAL(10,2),         -- 본사 부담
    
    -- 청구 정보
    invoice_number VARCHAR(50),
    invoice_date DATE,
    payment_status VARCHAR(20),    -- 'pending', 'paid', 'overdue'
    payment_date DATE,
    
    notes TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);
```

## 📈 귀책 협의 통계

### 귀책 유형별 통계

시스템에서 자동으로 다음 통계를 생성합니다:

```
• 귀책 유형별 비율
  - 생산처 귀책: X%
  - 제작처 귀책: Y%
  - 자연 마모: Z%
  - 기타: W%

• 협의 소요 시간
  - 1차 협의 평균 시간
  - 2차 협의 발생 비율
  - 최종 판정까지 평균 시간

• 비용 분석
  - 평균 수리 비용
  - 귀책 유형별 평균 비용
  - 월별/분기별 총 비용
```
